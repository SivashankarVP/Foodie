🚀 TOP STANDOUT & UNIQUE FEATURES
================================================================================

--------------------------------------------------------------------------------
1. 🤖 AI FOOD ASSISTANT CHATBOT (NATIVE CONVERSATIONAL ASSISTANT)
--------------------------------------------------------------------------------
- 💡 Concept: Embedded intelligent food ordering assistant directly integrated into 
  the ordering funnel rather than a generic third-party widget.
- ⚡ Differentiator:
  - 🧠 Intent-matching NLP engine processing queries for delivery policies, order tracking, 
    promo codes, restaurant details, and developer bio.
  - 💊 Interactive Quick-Action Pills enabling 1-click prompt execution.
  - 💬 Embedded floating UI (`chatbot.js` & `chatbot.css`) with smooth toggle transitions.

--------------------------------------------------------------------------------
2. 🎨 DYNAMIC MULTI-THEME SWITCHING (BURGER / PIZZA / FANTA MODES)
--------------------------------------------------------------------------------
- 💡 Concept: Real-time UI theme morphing engine tailored to different culinary moods.
- ⚡ Differentiator:
  - 🍔 Burger Theme: Warm Amber, Gold & Deep Charcoal palette.
  - 🍕 Pizza Theme: Rich Crimson Red, Cream White & Basil Accent palette.
  - 🍊 Fanta Theme: Vibrant Neon Orange, Electric Cyan & Dark Slate palette.
  - 🔄 Synchronizes CSS `:root` variables dynamically without requiring page reloads.
  - 💾 Theme state is persistently saved in `localStorage.setItem('cravego_theme', theme)`.

--------------------------------------------------------------------------------
3. 💎 BESPOKE GLASSMORPHISM UI DESIGN SYSTEM
--------------------------------------------------------------------------------
- 💡 Concept: Next-gen visual styling engineered from scratch with Vanilla CSS3.
- ⚡ Differentiator:
  - 🔮 Frosted glass containers utilizing `backdrop-filter: blur(12px)`.
  - 🌟 Layered depth effects with translucent borders, subtle ambient glows, and modern curvature.
  - 🎯 Zero reliance on heavy CSS frameworks (like Bootstrap), giving complete 
    design control, lightweight footprint, and blazing-fast render speeds.

--------------------------------------------------------------------------------
4. 🔒 SECURE LOGIN & AUTHENTICATION ROUTE GUARD
--------------------------------------------------------------------------------
- 💡 Concept: Multi-tenant user access control protecting sensitive ordering pages.
- ⚡ Differentiator:
  - 🛡️ Navigation Guard: Automatically checks session state before allowing access to 
    Checkout (`checkout.html`) or Order History (`order-history.html`).
  - 🔄 Smart Return URL: Redirects unauthenticated users to `login.html` and 
    seamlessly returns them to Checkout post-login.
  - 🔑 Database & JDBC Security: Uses PreparedStatements across Java Servlets (`LoginServlet`, 
    `RegisterServlet`) to prevent SQL Injection vulnerabilities.

--------------------------------------------------------------------------------
5. 🔔 SMART NON-BLOCKING TOAST NOTIFICATIONS
--------------------------------------------------------------------------------
- 💡 Concept: Modern notification system replacing disruptive native browser popups.
- ⚡ Differentiator:
  - ⚙️ Global Interceptor: Overrides default `window.alert()` inside `script.js`.
  - 🎯 Automatic Categorization: Evaluates message intent to trigger SUCCESS (Green ✅), 
    ERROR (Red ❌), or INFO (Amber ℹ️) Toast variants.
  - ⏱️ Animated Entry & Exit: Non-blocking UI popups with CSS slide-in keyframes 
    and 3-second auto-dismissal timers.

--------------------------------------------------------------------------------
6. 📜 USER-SPECIFIC ORDER HISTORY & STORAGE ISOLATION
--------------------------------------------------------------------------------
- 💡 Concept: Multi-tenant data segregation logic ensuring strict user privacy.
- ⚡ Differentiator:
  - 🔐 Isolated Accounts: Each order is stamped with the active user's email address.
  - 👁️ User-Specific View: `order-history.html` queries and displays only orders matching 
    the active session profile.
  - 🧼 Safe Teardown Logout: Clearing active sessions (`localStorage.removeItem('profile')`) 
    destroys session tokens while preserving user registration database intact.

--------------------------------------------------------------------------------
7. ✨ GSAP MICRO-ANIMATIONS & INTERACTIVE UX
--------------------------------------------------------------------------------
- 💡 Concept: High-end animation engine providing fluid UI responsiveness.
- ⚡ Differentiator:
  - 🎬 GreenSock Animation Platform (GSAP) integrations for hero section entrance.
  - 💫 Dynamic hover transformations, card lift-up effects, and button pulse keyframes.
  - 🌌 Interactive particle animations (`particles.js`) enhancing hero section aesthetics.

--------------------------------------------------------------------------------
8. 🍔 COMPLETE COMMERCIAL FOOD ORDERING FLOW
--------------------------------------------------------------------------------
- 💡 Concept: Real-world, multi-step food delivery pipeline matching Swiggy / Zomato standards.
- ⚡ Differentiator:
  - 🏢 Restaurant Catalog & Cuisine Filtering -> 🍕 Dish Selection & Custom Quantities ->
    🛒 Cart & Coupon Validation (`SIVA95`) -> 🔐 Auth Guard -> 🏡 Address Collection -> 
    💳 Payment Gateway Selection (UPI/Card/COD) -> 🧾 Invoice Generation & Order Tracking.
