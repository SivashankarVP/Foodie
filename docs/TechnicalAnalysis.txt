🛠️ SMART TECHNICAL IMPLEMENTATIONS
================================================================================

--------------------------------------------------------------------------------
1. 🔔 CUSTOM TOAST NOTIFICATION SYSTEM
--------------------------------------------------------------------------------
- Technical Mechanism: Global override of the native browser `window.alert()` inside `script.js`.
- Implementation Detail:
  - Intercepts alert calls and parses text strings to classify intent (Success, Error, Info).
  - Dynamically builds glassmorphic Toast DOM nodes with animated slide-in keyframes.
  - Automatically manages lifecycle timers (3-second display followed by fade-out teardown).
- Technical Benefit: Prevents disruptive browser popups and maintains non-blocking UI flow.

--------------------------------------------------------------------------------
2. 🤖 AI CHATBOT INTENT MATCHING
--------------------------------------------------------------------------------
- Technical Mechanism: Pattern-matching NLP evaluation engine inside `chatbot.js`.
- Implementation Detail:
  - Maps incoming user text inputs against structured intent dictionaries.
  - Uses regular expressions (`match(keywords)`) for fuzzy keyword parsing across 
    categories: Delivery Policies, Live Order Tracking, Promo Coupons (e.g. SIVA95), 
    and Developer Bio.
  - Includes fallback handling to keep user queries within the food platform domain.
- Technical Benefit: Delivers instant, context-aware responses without external API overhead.

--------------------------------------------------------------------------------
3. 🎨 THEME SWITCHING LOGIC
--------------------------------------------------------------------------------
- Technical Mechanism: Dynamic CSS root variable manipulation combined with state persistence.
- Implementation Detail:
  - `switchTheme(theme)` dynamically updates `:root` CSS design tokens for colors, 
    gradients, shadows, and hover states across Burger, Pizza, and Fanta themes.
  - Evaluates `window.location.pathname` to adjust hash navigation links (`index.html#home`), 
    ensuring seamless cross-page return transitions.
  - Persists selected theme across browser refreshes via `localStorage.setItem('cravego_theme', theme)`.
- Technical Benefit: Enables real-time visual morphing while preserving brand consistency.

--------------------------------------------------------------------------------
4. 🔐 USER-SPECIFIC LOCAL STORAGE
--------------------------------------------------------------------------------
- Technical Mechanism: Multi-tenant client storage isolation model in `script.js`.
- Implementation Detail:
  - Stamps newly generated orders with the active session user's email address (`userEmail`).
  - Queries and filters stored history array (`order_history`) against `activeProfile.email` 
    to enforce account privacy boundaries.
  - Implements targeted session teardown during logout (`localStorage.removeItem('profile')`), 
    destroying active session state without deleting persistent registered user databases.
- Technical Benefit: Provides enterprise-grade user privacy and persistent multi-account support.
